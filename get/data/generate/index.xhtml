What you need

It is easy to generate your own. All you need is:

  • An email address
  • A location: know you GPS area, OR...
  • A waypoint file - your own, a clubs, friends, OR...
  • Choose a location from the turnpoint database

You will be presented with a single file that you can use with any version of XCSoar from 5 up, that includes almost all the data that you need to use.
