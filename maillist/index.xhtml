User list

xcsoar-user (XCSoar Users) is the place for discussing current versions of XCSoar; how to install and use, tips and observations. Traffic is generally low-volume, but a digest is available.

Developer list

xcsoar-devel (XCSoar Development) is the place for discussing how to improve XCSoar. As any discussion here will be of features not yet implemented in XCSoar, there is no reason (other than curiosity) for most end-users to subscribe.
